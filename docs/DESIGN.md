# Flint - Design Document

> UI/UX specifications and visual design system for the Flint modding IDE.

---

## 1. Design Philosophy

### Core Principles

| Principle | Description |
|-----------|-------------|
| **Efficiency** | Every click should accomplish something meaningful |
| **Clarity** | Information hierarchy guides user attention |
| **Focus** | Remove clutter, show only what's needed for the current task |
| **Feedback** | Every action has visible confirmation |

### Design Goals

1. **Workflow-Centric** - UI stages match modding workflow
2. **Minimal Learning Curve** - Familiar patterns from IDEs and file managers
3. **Error Prevention** - Validate before destructive actions
4. **Progressive Disclosure** - Advanced features accessible but not overwhelming

## 4. Visual Design System

### 4.1 Color Palette

#### Dark Theme (Primary)

| Name | Hex | Usage |
|------|-----|-------|
| `--bg-primary` | `#0D1117` | Main background |
| `--bg-secondary` | `#161B22` | Panels, cards |
| `--bg-tertiary` | `#21262D` | Elevated surfaces |
| `--bg-hover` | `#30363D` | Hover states |
| `--border` | `#30363D` | Borders, dividers |
| `--text-primary` | `#E6EDF3` | Main text |
| `--text-secondary` | `#8B949E` | Muted text |
| `--text-muted` | `#6E7681` | Disabled, hints |

#### Accent Colors

| Name | Hex | Usage |
|------|-----|-------|
| `--accent-primary` | `#F97316` | Primary actions, Flint branding (orange) |
| `--accent-hover` | `#EA580C` | Primary hover |
| `--accent-blue` | `#58A6FF` | Links, selections |
| `--success` | `#3FB950` | Success states |
| `--warning` | `#D29922` | Warnings |
| `--error` | `#F85149` | Errors |

### 4.2 Typography

| Style | Font | Size | Weight | Line Height |
|-------|------|------|--------|-------------|
| H1 | Inter | 24px | 600 | 1.2 |
| H2 | Inter | 18px | 600 | 1.3 |
| H3 | Inter | 14px | 600 | 1.4 |
| Body | Inter | 14px | 400 | 1.5 |
| Small | Inter | 12px | 400 | 1.5 |
| Mono | JetBrains Mono | 13px | 400 | 1.4 |

### 4.3 Spacing Scale

```
4px  - Tight (inline elements)
8px  - Compact (list items)
12px - Base (component padding)
16px - Comfortable (section spacing)
24px - Relaxed (card padding)
32px - Spacious (major sections)
```

### 4.4 Border Radius

| Size | Value | Usage |
|------|-------|-------|
| Small | 4px | Buttons, inputs |
| Medium | 6px | Cards, dropdowns |
| Large | 8px | Modals, panels |

### 4.5 Shadows

```css
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.3);
--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.4);
--shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.5);
```
